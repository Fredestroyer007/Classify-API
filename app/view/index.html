API pour l'application Classify

Sélectionner un fichier audio et le pouvoir de l'intelligence artificielle va vous spécifiez le genre parmis les suivants : l'électronique, l'expérimental, le folk, le hip-hop, l'instrumental, l'international, le pop et le rock.

Veuillez prendre note que la librairie Librosa est affreusement lente et une prédiction peut prendre plusieurs dizaines de secondes pour un fichier audio assez volumineux.
